This page is about template pages. For instructions on how to use templates, see Help:Template.

A template is a Wikipedia page created to be included (transcluded) in other pages. Templates are used for message boxes, infoboxes, navigation boxes, and more. Templates help maintain consistency and avoid duplication of content.

Definition

A template is a page designed to be embedded in other pages using the syntax {{template name}}. Templates are reusable blocks of code or content used across multiple pages.

Uses of Templates

  • To display cleanup or maintenance messages
  • To standardize layouts using infoboxes
  • To create navigation panels for related pages
  • To reduce duplicated text or formatting

Template Syntax

To include a template on a page, use the double curly braces syntax:

{{TemplateName}}

Templates can accept parameters. For example:

{{Infobox person | name = John | birth_date = 1990 }}

Examples

  • {{Citation needed}} – adds a citation-needed notice
  • {{Stub}} – marks a short or incomplete article
  • {{Infobox}} – adds a structured table of information
